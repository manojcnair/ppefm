# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-02-08
### Added
- `ppefm_time_domain.m` MATLAB implementation of PPEFM (2008 coefficients + LT response).
- `ppefm_omni_example.m` example script that downloads OMNI 5‑minute data and runs PPEFM.
- `vdrift-model_jgr0499.f` original Scherliess & Fejer (1999) FORTRAN vertical drift model.
- `README.md` with usage, citations, and model description.
- `CITATION.cff` for repository citation metadata.
- `LICENSE` (CC BY‑NC 4.0).
- `images/example_plot.svg` illustrative output plot.
